# Plant monitor project

Project is meant to monitor plant growth and display it on a web page.

[Demo](https://www.plantmonitor.cloud)

Project is divided into 3 main parts:
- [server](https://github.com/pdelert/plant-server) - quarkus, auth0
- [front](https://github.com/pdelert/plant-front) - vue.js
- [monitor](https://github.com/pdelert/plant-monitor) - python
